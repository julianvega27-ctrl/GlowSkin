---

# 1. Información General del Proyecto

## Nombre del Proyecto
GlowSkin Store – Plataforma Web de Venta de Productos de Skincare

---

## Tipo de Proyecto
Aplicación web de comercio electrónico (E-Commerce).

---

## Descripción General
El proyecto consiste en el desarrollo de una plataforma web orientada a la venta de productos de skincare y cuidado de la piel.

La aplicación permitirá mostrar productos, gestionar pedidos, administrar inventario y facilitar compras online mediante una experiencia moderna, intuitiva y accesible desde cualquier dispositivo.

El sistema estará dirigido tanto a clientes finales como al administrador del negocio.

---

# 2. Metodología de Desarrollo

## żQué es SDD?

SDD (Specific Driven Development) es una metodología enfocada en construir software a partir de especificaciones claras, detalladas y estructuradas.

El objetivo principal es reducir ambigüedades antes de iniciar el desarrollo.

En esta metodología:

1. Primero se analiza el problema.
2. Luego se identifican necesidades.
3. Después se definen requisitos.
4. Finalmente se desarrolla el sistema.

---

## Beneficios de utilizar SDD

- Reduce errores de desarrollo.
- Evita retrabajo.
- Mejora la comunicación entre cliente y desarrollador.
- Facilita el uso de herramientas de IA.
- Permite documentar claramente el alcance del sistema.
- Facilita el mantenimiento futuro.
- Permite desarrollar funcionalidades de manera organizada.

# 3. Planteamiento del Problema

Actualmente muchos negocios de skincare realizan ventas únicamente mediante redes sociales o aplicaciones de mensajería.

Esto genera diversos problemas:

- Información desorganizada.
- Dificultad para mostrar todos los productos.
- Pérdida de clientes potenciales.
- Atención manual lenta.
- Mala experiencia de compra.
- Dificultad para gestionar pedidos.
- Falta de control de inventario.
- Problemas para gestionar pagos.
- Ausencia de automatización.
- Dependencia total de mensajes manuales.

Además, muchos clientes desean comprar rápidamente desde dispositivos móviles sin necesidad de contactar directamente al vendedor.
