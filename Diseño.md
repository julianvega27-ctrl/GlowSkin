# DESIGN SYSTEM DEL PROYECTO
# GlowSkin Store – Plataforma Web de Skincare
# Archivo de Referencia Visual para Desarrollo con IA

---

# 1. Objetivo del Documento

Este documento define el estilo visual, identidad gráfica, experiencia de usuario y lineamientos de diseño que deberán seguirse durante el desarrollo del sistema GlowSkin Store.

El objetivo es mantener:

- Consistencia visual.
- Diseño premium.
- Estética minimalista.
- Experiencia moderna.
- Compatibilidad con arquitectura ASP.NET Core MVC.
- Compatibilidad con SDD (Specific Driven Development).
- Claridad para desarrollo asistido por IA.

Este documento debe utilizarse como contexto permanente durante todo el desarrollo frontend.

---

# 2. Concepto General del Diseño

El sistema debe transmitir:

- Elegancia.
- Limpieza visual.
